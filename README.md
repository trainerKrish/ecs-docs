# Branching
- git branch - show the list of branches
- git branch <new-branch-name> - create a new branch
- git checkout <branch-name> - switch to the specified branch

# Merging
- // Switch to the target branch
- git merge <source-branch>

### Example
- git checkout main
- git merge feature
